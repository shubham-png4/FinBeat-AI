# FinBeat AI 🎵💰

An automated full-stack AI platform built using Python Flask that tracks daily micro-expenses, evaluates overall financial health with Google Gemini AI, and pulls context-aware matching soundscapes from the Spotify Web API.

---

## 🛠️ Tech Stack & Dependencies
* **Backend Framework:** Python 3.10+ & Flask
* **AI Model Engine:** Google Gemini API (`gemini-1.5-flash`)
* **Audio Pipeline API:** Spotify Web REST API (`spotipy`)
* **Interface Design:** HTML5, CSS3 Grid/Flexbox, Vanilla ES6 JavaScript

---
